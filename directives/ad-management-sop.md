# Ad Management SOP

## When to Use
Invoked whenever the user asks to create ads, manage ad campaigns, analyze ad performance, or anything related to paid advertising for clients.

## Platforms Covered
- Google Ads (Search, Display, YouTube)
- Meta Ads (Facebook + Instagram)
- TikTok Ads
- LinkedIn Ads (for B2B clients)

## Tools Required
- Ad platform knowledge and best practices
- Copywriting for ad creative
- Landing page creation (coordinate with website-builder agent)
- Analytics and reporting
- Budget calculator
- A/B testing framework

## Step-by-Step Process

### Step 1: Campaign Strategy
Gather from user or client file:
- [ ] Campaign goal (awareness, leads, sales, traffic)
- [ ] Target audience (demographics, interests, behaviors, location)
- [ ] Budget (daily/monthly)
- [ ] Timeline (start date, end date, ongoing)
- [ ] Offer or promotion (if any)
- [ ] Existing assets (images, videos, copy)
- [ ] Competitor analysis — what ads are competitors running?
- [ ] Landing page ready? (if not, coordinate with website-builder)

### Step 2: Audience Research
- Define primary audience persona
- Define secondary audience (for retargeting or lookalike)
- Research audience interests and behaviors on each platform
- Identify negative keywords/exclusions
- Map the customer journey (awareness → consideration → conversion)

### Step 3: Create Ad Content
For each campaign, create:

**Ad Copy Variations (minimum 3 per campaign):**
1. **Pain Point Angle** — Address the problem they're facing
2. **Benefit Angle** — Focus on the transformation/result
3. **Social Proof Angle** — Lead with testimonials or results

**Ad Structure:**
- **Headline** (max 40 chars for Google, flexible for Meta): Clear, benefit-driven
- **Description/Body:** Expand on the headline, include specificity
- **CTA:** Clear action (Get Started, Book Now, Learn More, Shop Now)
- **Visual Direction:** Describe what the image/video should convey

### Step 4: Campaign Setup
- Define campaign structure (campaigns → ad sets → ads)
- Set targeting parameters
- Set budget allocation
- Set bidding strategy (manual vs. automated)
- Set conversion tracking
- Create A/B test plan (test one variable at a time)

### Step 5: Launch Checklist
- [ ] Tracking pixel installed on landing page
- [ ] Conversion events properly configured
- [ ] Landing page is live and tested
- [ ] Ad copy proofread and approved
- [ ] Ad visuals meet platform specs
- [ ] Budget set correctly (daily/lifetime)
- [ ] Schedule set correctly
- [ ] UTM parameters added for tracking
- [ ] Negative keywords/exclusions in place

### Step 6: Monitor and Optimize
After launch, track daily for first week, then weekly:
- **Key Metrics:** CTR, CPC, CPM, CPA, ROAS, conversion rate
- **Actions:** Pause underperformers, scale winners, adjust targeting
- **Optimization Cadence:**
  - Days 1-3: Monitor, don't change (let algorithms learn)
  - Days 4-7: First optimizations based on data
  - Week 2+: Scale what works, kill what doesn't
  - Monthly: Full performance review and strategy adjustment

### Step 7: Reporting
Create client reports including:
- Campaign overview (spend, impressions, clicks, conversions)
- Key metrics vs. benchmarks
- What's working and why
- What needs improvement
- Recommendations for next period
- Save report to `memory/clients.md` under client section

## Budget Guidelines
| Monthly Budget | Recommended Platforms | Strategy |
|---|---|---|
| $500-1,000 | Pick ONE platform | Focus, test, iterate |
| $1,000-3,000 | 1-2 platforms | Split budget 70/30 |
| $3,000-10,000 | 2-3 platforms | Full funnel approach |
| $10,000+ | Multi-platform | Full funnel + retargeting |

## Ad Copy Frameworks

### PAS (Problem-Agitate-Solve)
- **Problem:** State the pain point
- **Agitate:** Make it feel urgent
- **Solve:** Present your solution

### AIDA (Attention-Interest-Desire-Action)
- **Attention:** Hook with bold claim or question
- **Interest:** Share relevant info
- **Desire:** Paint the picture of results
- **Action:** Clear CTA

### BAB (Before-After-Bridge)
- **Before:** Current painful state
- **After:** Desired future state
- **Bridge:** Your service is the bridge

## Quality Checks
- [ ] Ad copy has no spelling/grammar errors
- [ ] Visuals are high-quality and on-brand
- [ ] CTA is clear and compelling
- [ ] Landing page matches the ad message (message match)
- [ ] Tracking is properly set up before launch
- [ ] Budget is confirmed with user/client before spending
- [ ] Compliance with platform ad policies

## Notes
- Never spend client money without explicit approval
- Always start with a testing budget before scaling
- Document winning ad formulas in `memory/learnings.md`
- Save best-performing ads as templates
- Benchmark industry averages for each client's niche
